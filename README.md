# Sales Performance Tableau Dashboard

## Project Overview
This project presents an **interactive Tableau dashboard** built to analyze business performance using key metrics such as **Sales, Quantity, and Profit**.

The dashboard allows users to explore performance across different dimensions including **state, segment, shipping mode, product category, and sub-category**.

---

## Key Metrics
The dashboard focuses on three primary business metrics:

- **Sales**
- **Quantity**
- **Profit**

These metrics help evaluate overall business performance and product demand.

---

## Dashboard Features

### 1. Interactive Metric Selection
The dashboard includes **three interactive buttons**:

- Sales
- Quantity
- Profit

When a user clicks on any metric button:
- All charts dynamically update to reflect the selected metric.
- Users can quickly switch between different performance indicators.

---

### 2. State-wise Performance
A **horizontal bar chart** shows performance by **state**, helping identify top and low-performing regions.

---

### 3. Segment-wise Analysis
A chart analyzing performance across **customer segments**, enabling comparison between different customer groups.

---

### 4. Ship Mode Analysis
Visualization of metrics across different **shipping modes**, helping understand logistics performance.

---

### 5. Category-wise Analysis
Shows the contribution of different **product categories** to overall business performance.

---

### 6. Sub-Category Analysis
A detailed breakdown of **product sub-categories** to identify best-selling and low-performing product groups.

---

## Tools Used
- **Tableau** – Data visualization and dashboard creation
- **Dataset** – Retail sales dataset
